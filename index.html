WHAT IS

VERIFY?

An application for the Amazon Echo to help users set and check reminders

Verify is an Amazon Echo application that utilizes voice detection technology, allowing users to create, check, or remove a reminder, so users will never forget. Voice input from the user triggers our custom functions, which interact with the database and return customized responses.

Amazon Echo is a hands-free speaker you control with your voice. Echo connects to the Alexa Voice Service to play music, make calls, send and receive messages, provide information, news, sports scores, weather, and more—instantly. All you have to do is ask. When you want to use Echo, just say the wake word “Alexa” and Echo responds instantly.

“Alexa, open Verify”
“Alexa, set new Verify”
“Alexa, create new Verify”
“Alexa, when did I last Verify?“
“Alexa, remove last Verify.“
“Alexa, ask Verify - help!“
